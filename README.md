## Todos

CRUD app using Rails 5.2 for a Todo list app.

## Technologies
- The minimum Ruby version is 2.3
- RSpec - testing

## Usage
```
$ git clone git@github.com:alexscotttonge/RailsTodos
$ cd RailsTodos
$ bin/rails server
```

## Testing

Install ```rspec```

```gem install rspec```

Then run tests:
```$ rspec```

Test coverage: 98.5%
